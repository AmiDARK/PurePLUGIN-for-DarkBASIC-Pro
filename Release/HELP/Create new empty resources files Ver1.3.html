Resources Creation

If you need more resources files than the amount given with PurePLUGIN, you can freely follow the procedure:

Now, with the PureBASIC 4, you can directly add .rc files to your plugin via the PureBASIC compiler options.
a .rc file will =contain all resources
It is organised this way:

STRINGTABLE
{
1, "New Command 1%L%PPCommand1%"
2, "New Command 2%L%PPCommand2%"
}

STRINGTABLE
{
16, "New Command 3%L%PPCommand3%"
17, "New Command 4%L%PPCommand4%"
}

Note that you can add as many string table as you need.
The only important things to remember are these following statements:
- The 1st string table can contain up to 15 commands
- All string table (except 1st one) can contain up to 16 commands.
- Each commands must have a different id number
(for example string table Xcommands must not have the same id than string table Y commands)

Using this procedure, you can create a really large amount of Resources.
To calculate the numbers of the commands for each string table, simply use this:

FIRST_COMMAND = ( 16 * String Table ID ) ) - 15

You'll then have:
String Table 1 : commands 1-15
String Table 2 : commands 16-31
String Table 3 : commands 32-47
...
String Table 15 : commands 225-240
String Table 16 : commands 241-256
...
